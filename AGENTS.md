# Instructions for Oxyde Mogno project

## About
oxyde_mongo it is GUI Client for MongoDB, writen in Rust language, using GUI libraries Iced and iced_aw for additionals widgets.

## Development tips
* Ignore TODO.md - this file only for himan developer
* Do not inspect directory `notes` - it is only for human.

## Workflow Requirements
- After completing code changes, run `rustfmt`, then `cargo check`. If `cargo check` reports warnings or errors, fix them before finishing.

## Dependency Policy
- Do not change the versions of the `iced` and `iced_aw` dependencies in `Cargo.toml`.
- Ask for my approval before modifying any other existing dependencies.
- You may add new dependencies if they are required.

## Code Style
- Write all source code comments in English.

## Communication
- Communicate with me in Russian unless I ask otherwise.
